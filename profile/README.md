## 💡 Overview

<a href="#fathom">Fathom</a> is Seaweed Audio's spectral synthesis instrument designed for creating evolving textural sounds, deep soundscapes, and complex timbral movements — combining vector synthesis, wavetable playback, and spectral processing in an instrument optimized for ambient, cinematic, and experimental electronic music production.

The <a href="#fathom">vector synthesis architecture</a> is Fathom's core design — four oscillator sources simultaneously active, with a two-dimensional vector point controlling the blend between them in real time. Moving the vector point through the X-Y space morphs between the four sources, creating smooth timbral evolution as the vector position changes over time through automation, LFO modulation, or performance control. The evolving, morphing character that vector synthesis produces is distinctive — the sound continuously changes character as the vector moves rather than remaining at a static timbre. The <a href="#fathom">wavetable oscillator sources</a> draw waveforms from a library of wavetables — single-cycle waveforms that the oscillator scans through to produce evolving timbres beyond static waveforms. Each of the four vector sources can use different wavetables, and the vector morphing blends between the four simultaneously active wavetable positions for complex, continuously evolving sound character.

The <a href="#fathom">spectral processing section</a> manipulates the frequency content of the synthesized signal — frequency shifting, spectral blurring, harmonic filtering, and other spectral domain operations that produce the timbral transformations characteristic of Fathom's sound. Spectral manipulation goes beyond standard filter operations to reshape the harmonic structure of the sound in ways that conventional filters cannot achieve. The <a href="#fathom">modulation matrix</a> connects modulation sources — LFOs, envelopes, velocity, modwheel, MIDI controllers — to synthesis parameters through a configurable routing system that enables complex, multi-parameter modulation for the evolving, dynamic sounds that Fathom specializes in.

<a href="#fathom">Granular processing elements</a> layer granular synthesis techniques — time-stretching, pitch-shifting, and granular scatter — over the vector synthesis core, adding texture and density to the sound for the complex layered quality that ambient and cinematic music requires. <a href="#fathom">Effects chain</a> includes reverb, delay, chorus, distortion, and other effects processing within the instrument — creating complete, production-ready sounds without requiring external effects processing.

<a href="#fathom">Preset library</a> provides a starting point collection of Fathom sounds across different categories — pads, atmospheres, textures, leads, and other sound categories representing the range of Fathom's capabilities. <a href="#fathom">AU and VST plugin operation</a> runs Fathom as an Audio Units or VST instrument in Logic Pro, Ableton Live, and other compatible Mac DAWs.

Fathom's vector synthesis architecture produces the continuously evolving sound character that distinguishes it from static synthesizers. A conventional synthesizer plays a static sound until modulation changes it — the timbre remains constant unless LFOs or envelopes actively vary it. Vector synthesis makes timbral movement the fundamental character of the sound rather than an overlay — the vector position continuously blends between four sound sources, producing motion and evolution as an inherent property of the synthesis method rather than as applied modulation.

<summary>What is vector synthesis?</summary>

Vector synthesis places four sound sources at the corners of an X-Y grid, with a moveable vector point controlling the blend between them. Moving the point morphs between the four sources, creating evolving timbral change as the vector position shifts.
